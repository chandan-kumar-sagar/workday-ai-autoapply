# Demo Walkthrough

### Phase 1: Setup & Profile
1. Log in to the **React Dashboard**.
2. Navigate to **Resume Upload**.
3. Upload a sample PDF resume.
4. View the "Extracted Data" to verify AI parsing accuracy.

### Phase 2: Extension in Action
1. Open a **Workday Job Link** (e.g., `https://*.myworkdayjobs.com/...`).
2. Click the extension icon.
3. Click "Auto-Fill Form".
4. Watch as fields (Name, Email, Education, Previous Experience) are filled automatically.

### Phase 3: Tracking
1. Complete the application on the Workday site.
2. Return to the Dashboard **Applications** page.
3. Verify that the new application is listed with the "Applied" status.
